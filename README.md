# ⚡ Kinetic Focus

**Kinetic Focus** is a productivity dashboard web application designed to help users organize tasks, maintain focus sessions, and build consistent work habits through streak tracking and structured task prioritization.

🔗 **Live Demo:** https://kinetic-focus.netlify.app/

---

## 🚀 Features

* **Task Prioritization System**

  * Non-Negotiable Tasks
  * Task of the Day
  * Reschedule / Overflow Tasks

* **Focus Timer**

  * Customizable focus duration
  * Full-screen focus mode

* **Productivity Streak Tracking**

  * Visual fire animation to represent streaks
  * Tracks completed tasks

* **Drag-and-Drop Task Board**

  * Built using SortableJS
  * Move tasks between columns easily

* **User Authentication**

  * Login and Sign-Up interface
  * Credentials stored using browser local storage

* **Daily Journal**

  * Write reflections
  * Track gratitude entries

* **Modular Dashboard UI**

  * Single-page navigation
  * Clean dark theme interface

---
